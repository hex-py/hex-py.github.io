## 站点概况

本站采用了 Hexo 静态博客框架，主题为 NexT Gemini，托管在 Github page。除此之外，评论、统计等功能使用了以下服务：

* 评论系统：Valine + LeanCloud
* 访客统计：LeanCloud + 谷歌分析
* 静态资源：微博.
